# 学习 Learn

> 系统性学习Go语言核心概念，从基础语法到高级特性，构建扎实的理论基础

## 📚 学习路径

学习模块分为三个递进层次，帮助您全面掌握Go语言：

### 🔰 [基础语法 Fundamentals](/learn/fundamentals/)
Go语言的核心语法和基本概念，是所有Go开发的基石：
- 变量和数据类型
- 控制流程结构
- 函数和方法
- 接口和结构体
- 错误处理机制

### 🚀 [进阶特性 Advanced](/learn/advanced/)  
Go语言的高级特性和深层机制，提升开发效率：
- 并发编程（Goroutine & Channel）
- 反射和类型断言
- 包管理和模块系统
- 内存管理和垃圾回收
- CGO和系统调用

### 💡 [核心概念 Concepts](/learn/concepts/)
Go语言的设计哲学和底层原理，建立深度认知：
- Go的设计理念和哲学
- 类型系统和接口机制  
- 内存模型和并发安全
- 编译器原理和优化
- 运行时系统架构

## 🎯 学习建议

1. **顺序学习**：建议按照基础→进阶→概念的顺序学习
2. **实践为主**：每个知识点都要亲手编写代码验证
3. **循序渐进**：不急于求成，扎实掌握每个概念
4. **持续实践**：结合[工程实践模块](/practice/)巩固所学

---

🚀 **开始学习之旅**：选择适合您当前水平的模块开始学习吧！
